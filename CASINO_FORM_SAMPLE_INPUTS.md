## Common Issues

### Issue: "Template is required" error
**Solution:** Make sure you have created at least one "Casino Review Page Template" in the Templates section first.

### Issue: "Invalid input" error
**Solution:** Check browser console (F12) for specific field errors. All required fields must be filled.

### Issue: Form not saving
**Solution:** 
1. Check browser console for errors
2. Verify you're logged in as admin
3. Ensure all required fields have values
4. Check network tab to see API response
